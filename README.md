# 4X4X4 Cube Led Decoration.

## Requirement

### Hardwares

1. LED 4mm (64 nos)
2. 220 ohm resistor (Color Code: Red-Red-Brown)
3. Soldering kit 
4. Wires 

### Software

Arduino IDE
